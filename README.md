# Jungle Chess Reinforcement Learning

A reinforcement learning AI that plays Jungle Chess


## How to run

### Dependencies

This program requires maven

On first build:

`mvn install`

### Build and Run

To compile the code:

`mvn package`

To run:

`java -cp target/gs-maven-0.1.0.jar com.marcelfiore.jungle_ai.game.gui.GUI`
